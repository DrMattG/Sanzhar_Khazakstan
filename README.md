# Monitoring  brown bears in Khazakstan: A pilot study from the Altai Mountain Region

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.18848260.svg)](https://doi.org/10.5281/zenodo.18848260)

## Overview

This repository contains data and analysis code from a pilot camera-trap study assessing the presence and activity of brown bears in the Altai Mountain Region of Kazakhstan (2019-2023).

The study:

Deploys 10 camera traps (4,654 camera-days)

Documents seasonal and diel activity of brown bear (Ursus arctos)

Records 13 additional mammal species

Explores relationships between bear detections and:

- Elevation

- Distance to roads

- Distance to villages

This repository is intended to:

- Ensure full transparency and reproducibility

- Provide baseline biodiversity data for the Altai region

- Enable reuse for ecological modelling or meta-analysis

# Study species

The focal species is *Ursus arctos* in the Altai region which spans Kazakhstan, China, Mongolia and Russia, but little systematic monitoring has been conducted on the Kazakh side.

## Data description

| Variable        | Description                   |
| --------------- | ----------------------------- |
| Camera_ID       | Unique camera trap identifier |
| Date            | Date of capture               |
| Time            | Time of capture               |
| Species         | Common species name           |
| Elevation_m     | Elevation of camera location  |
| Dist_Road_km    | Distance to nearest main road |
| Dist_Village_km | Distance to nearest village   |


Independent detection events were defined as:

- ??? 6 hours apart for the same species at the same camera

For diel analyses:

- Maximum 1 detection per species per camera per hour

## Methods Summary

10 camera traps deployed in Kuchikha hunting estate (15 km˛)

Study period: 2019-2023

- Cameras mounted at ~80 cm height

- Year-round deployment

- Detection frequency was modelled using generalized linear models (negative binomial where appropriate) in R (v4.3.3).
